CSS COLOR GRADIENTS

CSS gradients mean display smooth transitions between two or more specified colors.

There are three types of gradients color :

• Linear Gradients (goes down/up/left/right/diagonally)
• Radial Gradients (defined by their center)
• Conic Gradients (rotated around a center point)

CSS Linear Gradients

To create a linear gradient you must define at least two color stops. You can also set a starting point and a direction (or an angle) along with the gradient effect. Syntax : - background-image: linear-gradient(direction, color-stop1, color-stop2, ...);

background-image: linear-gradient(red, yellow); (direction top to bottom (Default))


background-image: linear-gradient(to right, red, yellow); (to right, to bottom, to top, to left)


DIAGONAL : - background-image: linear-gradient(to top left, red, yellow); (to top left, to top right, to bottom left, to bottom right)


DEGREE : - background-image: linear-gradient(180deg, red, yellow);


multiple color : - background-image: linear-gradient(to left, red, yellow,blue,green);


REPEATING-LINEAR-GRADIENT COLOR : - background-image: repeating-linear-gradient(to left, red 5%, yellow 10%,green 15%);


REPEATING-LINEAR-GRADIENT COLOR : - background-image: repeating-linear-gradient( red , yellow 10%, green 15%);


REPEATING-LINEAR-GRADIENT COLOR : - background-image: repeating-linear-gradient(45deg, red , yellow 10%, green 15%);


REPEATING-LINEAR-GRADIENT COLOR : - background-image: repeating-linear-gradient(45deg,red, yellow 0, green 2%); height: 250px;
