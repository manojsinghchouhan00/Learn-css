CSS Image Reflections
The box-reflect property is used to create an image reflection.
The value of the 'box-reflect' property can be :- below, above, left , or right.


CSS object-fit property

This property is used to specify how an <img> or <video> should be resized to fit its container.

'object-fit' property can take one of the following values: (fill, contain, cover, none, scale-down);

img{
object-fit: cover;
}
